Compile
=======

Compile local script or query

Usage
-----

**dgms** **compile** [OPTIONS] <expression>...

.. list-table::
   :widths: 20 80
   :stub-columns: 1

   * - Parameters:
     - | <expression>...   
       | 	Script or query to compile

   * - Options:
     - | **-h**, **--help**
       | 	Show this help message and exit.
       | **-q**, **--query**
       |	Compile query expression
