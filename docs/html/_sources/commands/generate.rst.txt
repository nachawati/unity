Generate
========

Generate analytical artifacts

Usage
-----

**dgms** **generate** [OPTIONS] [COMMAND]

.. list-table::
   :widths: 20 80
   :stub-columns: 1

   * - Options:
     - | **-h**, **--help**
       | 	Show this help message and exit.

   * - Commands:
     - | **library**, **l**
       | 	Generate library module
       | **query**, **q**
       | 	Generate query module
